## 기능 구현
1. 실시간 시계: 
- 우측 메뉴바에서 로그인 시 출력

2. 로컬 스토리지를 사용한 로그인: 
- 우측 메뉴바

3. 로컬 스토리지를 사용한 투두리스트: 
- 달력과 동기화
- 추가, 삭제, 완료 표시

4. 랜덤 배경 이미지: 
- 메이플 닉네임 검색 시 적용

5. 날씨와 위치 (geolocation): 
- 우측 메뉴바에서 로그인 시 출력

6. CSS: 
- tailwind 사용

7. 메이플 닉네임 검색: 
- 메이플 공식 홈페이지에서 검색
- cors 우회 proxy 사용
- 영어 닉네임 대/소문자 불일치시 검색 안되는 버그 있음
- 해결 방법 아시는 분 도와주세요!

8. 일론 머스크 어록

9. 유튜브 플레이어: 
- 아이팟 디자인

10. AI 서비스 모음: 
- 우측 메뉴바

11. 메이플 타이머:
- 1분마다 설치기 알림
- 30분마다 경험치 쿠폰 알림
- 시작, 정지, 리셋, 30분 +/-
